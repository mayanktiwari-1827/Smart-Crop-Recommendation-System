# Smart Crop Recommendation System

A Machine Learning based web application that recommends the most suitable crop based on soil nutrients and environmental conditions.

## Features

* Crop Recommendation using Machine Learning
* Streamlit-based Interactive UI
* Multiple Model Comparison
* Real-time Predictions
* Trained Model Deployment using Pickle Files

## Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Joblib

## Input Parameters

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

## Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Model Training
4. Model Comparison
5. Model Serialization (.pkl)
6. Streamlit Application Development

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
